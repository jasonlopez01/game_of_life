Conway's Game of Life

This is an interactive version of Conway's Game of Life. Named for a British mathematician, John Horton Conway in 1970. The game posits a set of simple rules applied to a set of cells on a grid, after each "generation" the rules are applied and the inital pattern "evolves". This version uses Javascript to apply the rules and handle user input. It's all done as a single static webpage hosted via Github, code repo here. Click a cell to toggle it's state and make your own seep pattern, or select one from the dropdown that will develop over the generations. Some seed patterns will ociliate while others will reach a point of stablization. You can step a generation or let it run, the slider bar ajusts the speed. Note that the grid wraps, so that a pattern can move to the bottom or far right and re-appear at the top or far left.
